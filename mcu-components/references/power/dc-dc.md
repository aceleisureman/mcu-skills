# DC-DC 开关电源开发规范

## 1. 概述与选型指南

### 常见型号对比

| 型号 | 类型 | 输入范围 | 输出 | 最大电流 | 频率 | 特点 | 价格 |
|------|------|----------|------|----------|------|------|------|
| MP1584 | 降压 | 4.5~28V | 1.25~25V | 3A | 1.5MHz | 小封装、广泛使用 | ¥1~3 |
| LM2596 | 降压 | 4.5~40V | 1.25~37V | 3A | 150kHz | 经典、外置二极管 | ¥2~5 |
| TPS5430 | 降压 | 5.5~36V | 1.22~30V | 3A | 500kHz | 高效、内置 MOS | ¥3~8 |
| MP2315 | 降压 | 4.5~18V | 0.8~15V | 3A | 500kHz | 同步整流、高效 | ¥2~5 |
| SY8120 | 降压 | 4.5~18V | 0.6~15V | 2A | 1.4MHz | 小封装、廉价 | ¥0.5~2 |
| MT3608 | 升压 | 2~24V | 最高28V | 2A | 1.2MHz | 廉价升压 | ¥0.3~1 |
| XL6009 | 升压 | 3.6~32V | 最高38V | 4A | 400kHz | 大功率升压 | ¥1~3 |
| MP2307 | 降压 | 4.75~23V | 0.92~15V | 2A | 340kHz | 同步整流 | ¥2~5 |

### 选型建议
- **5V/12V→3.3V 大电流** → MP1584（通用降压，1.5MHz 小电感）
- **24V→5V 工业应用** → LM2596（耐压高，经典可靠）或 TPS5430
- **3.7V→5V 升压** → MT3608（廉价升压）
- **电池供电高效** → MP2315（同步整流，效率 >95%）
- **超小封装** → SY8120（SOT-23-6）

### 效率对比

```
LDO 效率: η = Vout / Vin × 100%
  例: 5V→3.3V → η = 66%, 34% 能量变热量

DC-DC 效率: η = 85~97% (取决于拓扑和负载)
  例: 5V→3.3V @300mA → DC-DC η ≈ 93%
  LDO 功耗: 0.51W → DC-DC 功耗: 0.17W
```

## 2. 硬件设计规范

### MP1584 降压电路

```
   Vin (12V)
     │
     ├── 22μF ──────┐  (输入电容, 陶瓷)
     │              │
     │         ┌────┴────┐
     ├─────────┤ IN    OUT├───┬─── Vout (3.3V)
     │         │  MP1584 │   │
     │     ┌───┤ EN      │  22μF (输出电容)
     │     │   │    GND  │   │
     │    100k  └────┬────┘  100nF
     │     │        │       │
     │     └────────┤       │
     │              │       │
     │      ┌───────┤       ├──┬── Vout
     │      │       │       │  │
     │   反馈分压   │      22μH  │
     │      │       │     电感  │
     │   ┌──┴──┐    │       │  │
     │   │ 上  │    │       │  │
     │   │电阻 │    │      ┌──┴──┐
     │   └──┬──┘    │      │续流 │ (肖特基)
     │      ├───────┤FB    │二极管│
     │   ┌──┴──┐    │      │SS34 │
     │   │ 下  │    │      └──┬──┘
     │   │电阻 │    │         │
     │   └──┬──┘    │         │
     │      │       │         │
    GND ────┴───────┴─────────┘

   反馈分压: Vout = 1.25 × (1 + R_top/R_bottom)
   Vout=3.3V: R_top=15kΩ, R_bottom=7.5kΩ (近似)
```

### 外围器件选型

#### 电感选型

```
电感值计算 (降压DC-DC):
L = (Vin - Vout) × Vout / (Vin × ΔIL × fsw)

ΔIL = 纹波电流, 通常取 Iout × 30%
fsw = 开关频率

例: Vin=12V, Vout=3.3V, Iout=2A, fsw=1.5MHz
ΔIL = 2 × 0.3 = 0.6A
L = (12-3.3) × 3.3 / (12 × 0.6 × 1.5M) = 2.66μH → 选 2.2μH 或 3.3μH

电感额定电流: > Iout + ΔIL/2 = 2 + 0.3 = 2.3A
→ 选饱和电流 ≥ 3A 的电感
```

#### 输出电容选型

```
输出纹波: ΔVout = ΔIL / (8 × fsw × Cout)

例: ΔIL=0.6A, fsw=1.5MHz, 目标 ΔVout < 20mV
Cout > 0.6 / (8 × 1.5M × 0.02) = 2.5μF → 选 22μF (留余量)

注意: 陶瓷电容的 DC bias 效应会降低实际容量
→ 22μF 6.3V X5R 在 3.3V 时实际约 12μF
→ 选用 2 颗 22μF 并联或 1 颗 47μF
```

### PCB 布局规范

```
DC-DC PCB 布局关键:

1. 输入电容紧贴 IC 输入引脚 (< 2mm)
2. 电感和续流二极管紧贴 SW 引脚
3. 输出电容紧贴电感输出端
4. 反馈走线远离 SW 节点 (避免干扰)
5. GND 大面积铺铜散热
6. 输入/输出走线足够宽 (1A → 0.3mm+)

    ┌────────┐
    │ INPUT  │── 22μF ──┬── IN ┌───┐
    │ CAP    │           │     │IC │
    └────────┘           │     │   │
                         │     │SW ├──┬── 电感 ──┬── OUT
                         │     │   │  │          │
                         │     │   │ 续流二极管  22μF
                         │     │   │  │          │
                         │     │GND│  │          │
                         │     └───┘  │          │
                         │            │          │
                         └────────────┴──────────┘
                              GND 大面积铺铜
```

## 3. 驱动开发规范

DC-DC 为硬件器件，但可编程 DC-DC（如 TPS62400）需要 I2C 配置：

```c
/**
 * @file programmable_dcdc.c
 * @brief 可编程 DC-DC 配置 (I2C 接口)
 */

/* 通过 I2C 设置输出电压 */
dcdc_status_t dcdc_set_voltage(i2c_config_t *i2c, uint8_t addr,
                                uint16_t mv) {
    uint8_t cmd[3];

    /* 电压转换为寄存器值 (依芯片而定) */
    /* 示例: TPS62400, 0.625V~3.3V, 步进 25mV */
    uint16_t reg_val = (mv - 625) / 25;
    if (reg_val > 0x7F) return DCDC_ERR_INVALID_VOLT;

    cmd[0] = 0x01;  /* VOUT 寄存器地址 */
    cmd[1] = reg_val & 0xFF;

    if (i2c_write(i2c, addr, cmd, 2) != 0) {
        return DCDC_ERR_COMM;
    }
    return DCDC_OK;
}

/* 读取 DC-DC 状态 */
dcdc_status_t dcdc_read_status(i2c_config_t *i2c, uint8_t addr,
                                uint8_t *status) {
    uint8_t reg = 0x00;  /* STATUS 寄存器 */
    if (i2c_write_read(i2c, addr, &reg, 1, status, 1) != 0) {
        return DCDC_ERR_COMM;
    }
    return DCDC_OK;
}

/* 动态电压调节 (DVFS - 动态电压频率调节) */
void dcdc_dvfs_adjust(cpu_load_t load) {
    switch (load) {
        case CPU_LOAD_IDLE:
            dcdc_set_voltage(&i2c_cfg, DCDC_ADDR, 900);  /* 0.9V */
            cpu_set_freq(48);  /* 48MHz */
            break;
        case CPU_LOAD_NORMAL:
            dcdc_set_voltage(&i2c_cfg, DCDC_ADDR, 1100); /* 1.1V */
            cpu_set_freq(96);  /* 96MHz */
            break;
        case CPU_LOAD_HIGH:
            dcdc_set_voltage(&i2c_cfg, DCDC_ADDR, 1300); /* 1.3V */
            cpu_set_freq(168); /* 168MHz */
            break;
    }
}
```

## 4. 调试与测试规范

### 验证清单
- [ ] 输入电压在 IC 耐压范围内
- [ ] 电感饱和电流 > 最大负载电流 × 1.3
- [ ] 输入/输出电容容量和耐压满足要求
- [ ] 续流二极管为肖特基（正向压降低、反向恢复快）
- [ ] 反馈分压电阻精度 ≤ 1%
- [ ] PCB 布局符合规范（短回路、大面积 GND）

### 测试方法
- **示波器测纹波**：输出纹波应 < 50mV
- **负载调整率**：负载从 10%→100% 变化时输出电压变化率 < 1%
- **线性调整率**：输入电压变化时输出电压变化率 < 0.5%
- **效率测试**：不同负载下测量输入/输出功率，计算效率
- **瞬态响应**：负载阶跃时观察电压恢复时间
- **温度测试**：满载运行下 IC 和电感温度 < 85°C

## 5. 常见问题与避坑指南

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 输出电压不准 | 反馈分压电阻精度差 | 用 1% 精密电阻 |
| 输出纹波大 | 输出电容不足/ESR高 | 加大容量，用低ESR陶瓷电容 |
| 电感啸叫 | 进入 DCM 模式/电感饱和 | 增大电感量或换高饱和电流电感 |
| 效率低 | 续流二极管正向压降大 | 换肖特基或用同步整流DC-DC |
| 发热严重 | PCB 散热不足 | 增大铺铜面积，加散热过孔 |
| 输出振荡 | 补偿网络不对 | 调整 Type II/III 补偿参数 |
| 启动失败 | 软启动时间不足 | 增大 SS 电容 |
| 轻载纹波大 | DCM 模式纹波大 | 选 PSM 模式或强制 CCM |
| 电磁干扰大 | 开关频率高/布局差 | 加 RC 缓冲器、优化布局 |
| 电感烧毁 | 饱和电流不足 | 选 Isat > 最大负载 × 1.3 |
