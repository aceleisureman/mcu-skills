# 通用硬件设计规范

## 1. 电源系统设计

### 1.1 多路电源设计原则

MCU 系统通常需要多路电源，设计时需遵循以下原则：

| 电源轨 | 典型电压 | 用途 | 常用方案 |
|--------|----------|------|----------|
| 主电源 | 5V / 12V / 24V | 外部输入 | DC-DC 降压或适配器 |
| 数字电源 | 3.3V | MCU、传感器、数字 IC | LDO (AMS1117-3.3 / HT7333) |
| 模拟电源 | 3.3V (独立) | ADC、DAC、传感器模拟端 | 独立 LDO + 磁珠隔离 |
| 外设电源 | 5V / 12V | 电机、继电器、显示屏背光 | 独立 DC-DC |
| 电池电源 | 3.0~4.2V | 锂电池供电系统 | 充电 IC + 保护 IC |

**设计要点**：
- 数字电源和模拟电源必须分开供电，通过磁珠（如 600Ω@100MHz）单点连接
- 大功率外设（电机、加热器）必须独立电源，不可与 MCU 共用
- 电源输入端需有防反接保护（二极管 / MOSFET）
- 多板系统需考虑电源连接器的载流能力和接触电阻

### 1.2 上电时序

多电源系统中，上电时序至关重要：

```
推荐上电顺序:
  1. 主电源 (5V/12V)
  2. 数字电源 (3.3V) —— 稳定后
  3. 模拟电源 (3.3V_A) —— 可与数字电源同时或稍后
  4. 外设使能 —— MCU 启动完成后

禁止:
  - I/O 信号先于 MCU 电源上电 (可能导致闩锁效应)
  - 模拟电源先于数字电源 (可能导致 ADC 读数异常)
```

**时序控制方法**：
- 使用带 EN 引脚的 LDO/DC-DC，通过 RC 延时控制使能
- 使用电源时序控制 IC (如 TPS3850)
- MCU 控制外设电源使能，启动后按序打开

### 1.3 功耗估算

```
总功耗 = MCU功耗 + 外设功耗 + 驱动电路功耗 + 电源转换损耗

电源转换效率: η = P_out / P_in
DC-DC 效率: 85%~95%
LDO 效率: V_out / V_in (如 3.3/5.0 = 66%)
LDO 热耗散: P_loss = (V_in - V_out) * I_load

示例: 5V -> 3.3V @ 200mA
  LDO: P_loss = (5-3.3) * 0.2 = 0.34W, 需检查封装热阻
  DC-DC: P_loss = 3.3*0.2*(1/0.9 - 1) = 0.073W
```

**散热设计**：
- 芯片结温 T_j = T_a + P * θ_ja (环境温度 + 功耗 * 结到环境热阻)
- SOT-23 封装 θ_ja ≈ 250°C/W, SOIC-8 ≈ 150°C/W, QFN ≈ 40°C/W
- T_j 必须低于数据手册最大值 (通常 125°C 或 150°C)

## 2. 去耦电容规范

### 2.1 基本原则

每个 IC 的电源引脚必须就近放置去耦电容：

| 电容类型 | 容值 | 作用 | 放置位置 |
|----------|------|------|----------|
| 陶瓷电容 | 100nF (0.1μF) | 滤除高频噪声 | 紧贴 IC VCC 引脚 (< 5mm) |
| 陶瓷/钽电容 | 10μF | 滤除低频纹波/储能 | IC 附近 (< 20mm) |
| 陶瓷电容 | 10nF | 滤除更高频噪声 | 高速 IC 旁 |

### 2.2 各类 IC 去耦方案

```
MCU (STM32/ESP32):
  每组 VDD/VSS: 100nF + 一个全局 10μF
  VDDA/VSSA: 100nF + 1μF + 10μF (磁珠隔离)

传感器 (I2C/SPI):
  VCC: 100nF + 10μF

运算放大器:
  每个电源引脚: 100nF, 双电源各加 1μF

DC-DC:
  输入: 10μF~47μF (低ESR陶瓷)
  输出: 22μF~100μF (低ESR陶瓷 + 钽电容)

晶振:
  不需要去耦电容, 但电源引脚需 100nF
```

### 2.3 电容选型注意

- **材质**：高频去耦选 X7R 或 X5R (温度稳定), 不选 Y5V/Z5U
- **耐压**：至少为工作电压的 2 倍 (3.3V 系统选 6.3V 或 10V)
- **ESR**：DC-DC 输出电容 ESR 影响环路稳定性, 按芯片手册推荐选型
- **直流偏压效应**：陶瓷电容实际容值随直流偏压降低, 10μF/10V 在 3.3V 偏压下可能只有 6~7μF

## 3. PCB 布局规范

### 3.1 信号完整性

**高速信号 (CLK, SPI, UART > 1MHz)**：
- 走线尽量短而直, 避免直角走线 (用 45° 或圆弧)
- 时钟线远离敏感信号 (ADC 输入、复位)
- 差分信号等长走线, 误差 < 5mil
- 高速信号线下方保持完整地平面

**晶振布局**：
- 尽可能靠近 MCU 晶振引脚 (< 5mm)
- 走线尽量短, 不跨分割平面
- 晶振下方禁止走其他信号
- 晶振周围用地线包围 (Guard Ring), 接地到 MCU 地

### 3.2 电源完整性

- 电源走线尽量短而宽, 减少寄生电感
- 电源和地平面尽量完整, 避免分割
- 大电流路径 (电源输入 → DC-DC → 负载) 走线要宽
- 过孔会引入寄生电感 (~0.5nH/via), 大电流用多个过孔并联

**电源走线宽度参考** (1oz 铜, 温升 10°C)：

| 电流 | 最小线宽 (外层) | 最小线宽 (内层) |
|------|-----------------|-----------------|
| 0.5A | 0.2mm (8mil) | 0.4mm (16mil) |
| 1A | 0.4mm (16mil) | 0.8mm (32mil) |
| 2A | 0.8mm (32mil) | 1.5mm (60mil) |
| 3A | 1.2mm (48mil) | 2.5mm (100mil) |
| 5A | 2.0mm (80mil) | 4.0mm (157mil) |

### 3.3 地平面设计

- 尽量使用完整的地平面 (多层板至少一层为 GND)
- 模拟地和数字地分开铺铜, 单点连接 (磁珠或 0Ω 电阻)
- 避免地平面分割导致回流路径不连续
- 连接器附近的地要完整, 提供低阻抗回流路径
- 混合信号 IC (如 MCU) 的 AGND 和 DGND 在芯片下方连接

### 3.4 元器件布局原则

1. **按功能分区**：电源区、MCU 区、传感器区、驱动区物理分离
2. **去耦电容就近**：100nF 去耦电容紧贴 IC 电源引脚, 走线 < 5mm
3. **晶振远离干扰**：远离 DC-DC、大电流走线、高速时钟
4. **模拟远离数字**：ADC 输入引脚远离数字信号走线
5. **发热器件靠边**：LDO、功率 MOSFET 靠近板边, 利于散热
6. **接口器件靠边**：连接器、USB、电源接口在板边缘
7. **高矮搭配**：高的元件 (电解电容、连接器) 不影响矮元件焊接

## 4. 走线规范

### 4.1 一般走线规则

| 信号类型 | 线宽 | 线距 | 说明 |
|----------|------|------|------|
| 电源线 | 0.5~2.0mm | 0.3mm | 按电流计算 |
| 地线 | 0.5~1.0mm | 0.3mm | 尽量宽 |
| 普通信号 | 0.15~0.25mm | 0.15~0.2mm | 6~10mil |
| 高速时钟 | 0.2mm | > 0.3mm | 远离敏感信号 |
| 差分信号 | 0.15mm | 0.15mm (对内) | 等长、等距 |

### 4.2 差分信号

- 差分对内两根线等长 (误差 < 5mil)
- 差分对间距一致, 阻抗匹配 (USB 90Ω, CAN 120Ω, RS485 120Ω)
- 差分对之间间距 > 3 倍线宽
- 尽量少打过孔, 必须打孔时差分对一起打

### 4.3 高速信号

- SPI 时钟 > 10MHz 时需考虑阻抗匹配和走线长度
- USB 差分线阻抗 90Ω ±10%
- 高速信号走线避免 stub (分支)
- 信号换层时, 旁边打地过孔提供回流路径

## 5. 接口防护设计

### 5.1 ESD 防护

| 接口类型 | ESD 器件 | 放置位置 |
|----------|----------|----------|
| USB | USBLC6-2SC6 | 连接器到 MCU 之间 |
| RS485 | PESD3V3L5UY | A/B 线对地 |
| I2C (板外) | ESDA6V1-1U2 | SCL/SDA 对地 |
| 按钮/按键 | 100nF 对地 + 限流电阻 | GPIO 入口 |
| 天线 | ESD 到地 | RF 路径 |

**选型原则**：
- 工作电压 > 信号最高电压 (3.3V 系统选 5V 或 6V)
- 钳位电压 < 被保护器件最大耐压
- 结电容 < 5pF (高速信号), 普通信号 < 30pF

### 5.2 过流保护

- **自恢复保险丝 (PTC)**：用于 USB、电机驱动, 过流自动断开, 恢复后自愈合
- **限流电阻**：LED 限流、IO 驱动限流
- **DC-DC 限流**：DC-DC IC 内置限流, 需按手册设置电流检测电阻

### 5.3 过压保护

- **TVS 二极管**：瞬态过压保护, 响应速度 < 1ps
- **稳压二极管**：持续过压保护 (功耗较大)
- **OVP IC**：专用过压保护芯片 (如 LM5050)

## 6. EMC 设计要点

### 6.1 辐射发射控制

- **时钟信号**：最大的辐射源, 走线尽量短, 下方完整地平面, 必要时加 RC 滤波
- **开关电源**：DC-DC 开关频率谐波, 输入输出加 LC 滤波, 电感选屏蔽型
- **高速总线**：SPI/I2C 限速, 加串联电阻 (22~100Ω) 减缓边沿
- **地平面**：保持完整, 减少回流环路面积
- **屏蔽**：金属外壳接大地, 连接器处做好 360° 屏蔽接地

### 6.2 抗干扰设计

- **复位脚**：接 100nF 到地 + 10kΩ 上拉, 防止噪声复位
- **未用引脚**：配置为输出低或上拉/下拉, 不可悬空
- **模拟输入**：RC 低通滤波 (1kΩ + 100nF)
- **走线**：敏感信号 (ADC) 远离噪声源 (DC-DC, 时钟)
- **地线环路**：避免形成大地环路, 单点接地

### 6.3 静电防护

- 外露接口必须加 ESD 保护
- PCB 设计中 ESD 器件放在连接器最前端
- ESD 泄放路径直接到地平面, 不经过敏感器件
- 按键、旋钮等人体接触部位加 ESD 保护

## 7. 常用接口电平匹配方案

| 驱动端 | 接收端 | 匹配方案 | 说明 |
|--------|--------|----------|------|
| 5V MCU | 3.3V 器件 | 电阻分压 / 电平转换 IC | 74LVC245, TXS0108E |
| 3.3V MCU | 5V 器件 | 直接连接 (若 5V 器件 VIH ≤ 2V) | 或使用电平转换 |
| 3.3V MCU | 5V I2C | 电平转换 IC | PCA9306, TXS0102 |
| MCU GPIO | RS485 | MAX485 / SP3485 | 差分电平转换 |
| MCU GPIO | CAN | TJA1050 / MCP2551 | 差分电平转换 |
| MCU UART | RS232 | MAX3232 | 电平转换 + 反相 |
| 3.3V ADC | ±10V 信号 | 运放衰减电路 | 差分放大器 |
| MCU GPIO | 继电器 | 三极管 + 续流二极管 | 电流驱动 + 反电动势保护 |

**电平转换 IC 选型**：
- **双向自动方向**：TXS0108E (I2C/SPI), TXB0108 (推挽)
- **单向**：74LVC245 (缓冲), SN74LVC1T45 (单通道)
- **I2C 专用**：PCA9306, TCA9406
- **多电压**：MAX3378 (1.2V~5.5V)

## 8. 设计审查清单

### 原理图审查
- [ ] 电源电压等级和容量是否满足需求
- [ ] 每个 IC 电源引脚有去耦电容
- [ ] 模拟/数字电源隔离
- [ ] ESD/TVS/过流保护已设计
- [ ] 上电时序是否正确
- [ ] 未用引脚有明确处理 (上下拉或模拟)
- [ ] 晶振负载电容匹配
- [ ] 上拉/下拉电阻阻值合理

### PCB 审查
- [ ] 去耦电容紧贴 IC 电源引脚
- [ ] 电源走线宽度满足电流要求
- [ ] 地平面完整, 无不必要分割
- [ ] 模拟/数字地单点连接
- [ ] 高速信号下方有完整参考地
- [ ] 晶振周围有地保护环
- [ ] 接口 ESD 器件在连接器最前端
- [ ] 丝印清晰, 元件位号/极性标注完整
- [ ] 测试点充足 (关键信号、电源、地)
- [ ] DRC (设计规则检查) 无错误
